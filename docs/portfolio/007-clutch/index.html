Robust and easy-integrating electronic clutch system for a YFZ450R engine, using cheap drill gearboxes and a clever over-center mechanism.
Download the Design Docs

The design was driven by a few desires:

  1. Minimize modifications to the engine.
  2. Be as bulletproof as possible in function, making it impossible to over or under extend and damage the clutch.
  3. Require no supplied power to maintain the clutch actuator at fully engaged and fully decoupled states.
  4. Allow for variable positioning for smooth clutch release and vehicle launch.
  5. Allow for adjustments to be done mechanically rather than requiring software intervention.

This mechanism was exactly perfect for the job - and required very unsophisticated parts:

  • A cheap drill gearmotor
  • A few custom bearing blocks
  • A tie rod
  • A throttle position sensor for position feedback.
  • The existing engine-side clutch lever which would normally be actuated by a pull cable from handlebars on the quad bike. And it worked beautifully. I’ve never seen such a well-executed clutch mechanism. Most are complicated linear actuators. This one just works.
Development
RoseGPE has wanted an electronic clutch for a long time. Part of this is to assist in electronic launch control, part of this is to reduce need for routing pull cables, but mostly, it’s to keep the driver’s hands on the wheel, and provide some safety features.
